Websites To Change:

Domain

Name replacement will run on the domains listed here. Note that you may have to list a given website more than once. For example, wikipedia.org and en.wikipedia.org are different domains. Note also that the "www" which appears in many website links is important. Here, www.time.gov and time.gov are considered different domains. This is a little tedious, and could possibly be streamlined in the future, but for now it is the most reliable way to follow your preferences correctly.

New Names:

Old Name New Name Match whole name only Case sensitive Match all caps, all lower-case & 1st letter capitalized.

To replace the name "Lila" with the name "Marcello", write "Lila" in the first column and "Marcello" in the second. The 3rd column is checked by default. Unchecking it would result in "Lilac" being replaced with "Marcelloc" in this scenario. The 4th column is unchecked by default. Checking it would result in "lila" not being replaced, since "lila" does not match "Lila". The 5th column is checked by default. By default "LILA" would be replaced with "MARCELLO", and "lila" with "marcello".
